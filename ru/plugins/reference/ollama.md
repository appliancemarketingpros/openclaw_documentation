# Plugin Ollama

Добавляет в OpenClaw поддержку провайдеров моделей Ollama и Ollama Cloud.

## Распространение

  * Пакет: `@openclaw/ollama-provider`
  * Способ установки: включен в OpenClaw


## Интерфейс

провайдеры: ollama, ollama-cloud; контракты: memoryEmbeddingProviders, webSearchProviders
