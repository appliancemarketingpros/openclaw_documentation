# [TOOLS.md](<http://TOOLS.md>) \- บันทึกภายในเครื่อง

Skills กำหนด _วิธี_ การทำงานของเครื่องมือ ไฟล์นี้มีไว้สำหรับ _รายละเอียดเฉพาะของคุณ_ — สิ่งที่เป็นเอกลักษณ์ของการตั้งค่าของคุณ

## สิ่งที่ควรใส่ไว้ที่นี่

สิ่งต่างๆ เช่น:

  * ชื่อและตำแหน่งของกล้อง
  * โฮสต์และนามแฝง SSH
  * เสียงที่ต้องการสำหรับ TTS
  * ชื่อลำโพง/ห้อง
  * ชื่อเล่นของอุปกรณ์
  * สิ่งใดก็ตามที่ขึ้นอยู่กับสภาพแวดล้อมเฉพาะ

## ทำไมต้องแยกไว้ต่างหาก?

Skills ใช้ร่วมกันได้ การตั้งค่าของคุณเป็นของคุณเอง การแยกออกจากกันแบบนี้ช่วยให้คุณอัปเดต Skills ได้โดยไม่ทำให้บันทึกของคุณหายไป และแชร์ Skills ได้โดยไม่เปิดเผยโครงสร้างพื้นฐานของคุณ

* * *

เพิ่มอะไรก็ได้ที่ช่วยให้คุณทำงานได้สะดวกขึ้น นี่คือโพยสรุปของคุณ
