# File Transfer Plugin

透過專用節點命令，在已配對的節點上擷取、列出及寫入檔案。對於最大 16 MB 的二進位檔，使用 node.invoke 透過 base64 傳輸，以避開 bash stdout 截斷。

## 發行

  * 套件：`@openclaw/file-transfer`
  * 安裝路徑：包含於 OpenClaw
