# `openclaw clawbot`

Застарілий простір назв псевдонімів, збережений для зворотної сумісності.

Поточний підтримуваний псевдонім:

  * `openclaw clawbot qr` (така сама поведінка, як у [`openclaw qr`](</uk/cli/qr>))


## Міграція

Надавайте перевагу сучасним командам верхнього рівня напряму:

  * `openclaw clawbot qr` -> `openclaw qr`
