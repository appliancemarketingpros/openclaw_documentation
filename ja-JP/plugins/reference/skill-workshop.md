# Skill Workshop Plugin

反復可能なワークフローをワークスペーススキルとして取り込み、保留中のレビュー、安全な書き込み、スキルプロンプトの更新に対応します。

## 配布

  * パッケージ: `@openclaw/skill-workshop`
  * インストール経路: OpenClaw に含まれる
