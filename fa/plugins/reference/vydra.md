# Plugin Vydra

پشتیبانی از ارائه‌دهنده مدل Vydra را به OpenClaw اضافه می‌کند.

## توزیع

  * بسته: `@openclaw/vydra-provider`
  * مسیر نصب: در OpenClaw گنجانده شده است


## سطح

ارائه‌دهندگان: vydra؛ قراردادها: imageGenerationProviders، speechProviders، videoGenerationProviders
