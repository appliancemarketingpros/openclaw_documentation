# Tencent Plugin

پشتیبانی از ارائه‌دهندهٔ مدل Tencent TokenHub را به OpenClaw اضافه می‌کند.

## توزیع

  * بسته: `@openclaw/tencent-provider`
  * مسیر نصب: در OpenClaw گنجانده شده است


## سطح

providers: tencent-tokenhub
