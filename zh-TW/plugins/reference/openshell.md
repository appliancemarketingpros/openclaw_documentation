# Openshell Plugin

由 OpenShell 提供支援的沙箱後端，具備鏡像的本機工作區與以 SSH 為基礎的命令執行。

## 發行方式

  * 套件：`@openclaw/openshell-sandbox`
  * 安裝途徑：已包含於 OpenClaw
