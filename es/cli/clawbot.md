# `openclaw clawbot`

Espacio de nombres de alias heredado mantenido por compatibilidad con versiones anteriores.

Alias compatible actual:

  * `openclaw clawbot qr` (mismo comportamiento que [`openclaw qr`](</es/cli/qr>))


## Migración

Prefiere usar directamente los comandos modernos de nivel superior:

  * `openclaw clawbot qr` -> `openclaw qr`
