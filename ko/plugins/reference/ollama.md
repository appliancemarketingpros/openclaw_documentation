# Ollama Plugin

OpenClaw에 Ollama 모델 제공자 지원을 추가합니다.

## 배포

  * 패키지: `@openclaw/ollama-provider`
  * 설치 경로: OpenClaw에 포함됨


## 노출 영역

제공자: ollama; 계약: memoryEmbeddingProviders, webSearchProviders
