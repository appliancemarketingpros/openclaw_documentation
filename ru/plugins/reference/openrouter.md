# Plugin OpenRouter

Добавляет поддержку провайдера моделей OpenRouter в OpenClaw.

## Распространение

  * Пакет: `@openclaw/openrouter-provider`
  * Способ установки: включен в OpenClaw


## Интерфейс

providers: openrouter; contracts: imageGenerationProviders, mediaUnderstandingProviders, musicGenerationProviders, speechProviders, videoGenerationProviders
