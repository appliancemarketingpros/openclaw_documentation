# 文件传输插件

通过专用节点命令在已配对节点上获取、列出和写入文件。通过对最大 16 MB 的二进制文件使用基于 `node.invoke` 的 base64，绕过 bash stdout 截断。

## 分发

  * 包：`@openclaw/file-transfer`
  * 安装路径：包含在 OpenClaw 中
