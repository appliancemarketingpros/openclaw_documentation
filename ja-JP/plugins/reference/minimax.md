# MiniMax Plugin

OpenClaw に MiniMax、MiniMax Portal モデルプロバイダーのサポートを追加します。

## 配布

  * パッケージ: `@openclaw/minimax-provider`
  * インストール経路: OpenClaw に含まれています


## 公開面

プロバイダー: minimax, minimax-portal; コントラクト: imageGenerationProviders, mediaUnderstandingProviders, musicGenerationProviders, speechProviders, videoGenerationProviders, webSearchProviders
