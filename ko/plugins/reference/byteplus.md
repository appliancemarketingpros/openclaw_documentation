# BytePlus Plugin

OpenClaw에 BytePlus 및 BytePlus Plan 모델 제공자 지원을 추가합니다.

## 배포

  * 패키지: `@openclaw/byteplus-provider`
  * 설치 경로: OpenClaw에 포함됨


## 노출 영역

providers: byteplus, byteplus-plan; contracts: videoGenerationProviders
