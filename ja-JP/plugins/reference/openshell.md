# Openshell Plugin

OpenShell を利用し、ミラーされたローカルワークスペースと SSH ベースのコマンド実行を備えたサンドボックスバックエンド。

## 配布

  * パッケージ: `@openclaw/openshell-sandbox`
  * インストール経路: OpenClaw に含まれる
