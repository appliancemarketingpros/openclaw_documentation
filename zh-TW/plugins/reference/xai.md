# xAI Plugin

為 OpenClaw 新增 xAI 模型提供者支援。

## 發行

  * 套件：`@openclaw/xai-plugin`
  * 安裝路徑：包含於 OpenClaw


## 介面

提供者：xai；合約：imageGenerationProviders, mediaUnderstandingProviders, realtimeTranscriptionProviders, speechProviders, tools, videoGenerationProviders, webSearchProviders
