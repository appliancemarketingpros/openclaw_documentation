# MiniMax Plugin

เพิ่มการรองรับผู้ให้บริการโมเดล MiniMax, MiniMax Portal ให้กับ OpenClaw

## การแจกจ่าย

  * แพ็กเกจ: `@openclaw/minimax-provider`
  * เส้นทางการติดตั้ง: รวมอยู่ใน OpenClaw


## พื้นผิวการใช้งาน

ผู้ให้บริการ: minimax, minimax-portal; สัญญา: imageGenerationProviders, mediaUnderstandingProviders, musicGenerationProviders, speechProviders, videoGenerationProviders, webSearchProviders
