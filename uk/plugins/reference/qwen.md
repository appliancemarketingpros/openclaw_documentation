# Qwen Plugin

Додає до OpenClaw підтримку провайдерів моделей Qwen, Qwen Cloud, Model Studio та DashScope.

## Дистрибуція

  * Пакет: `@openclaw/qwen-provider`
  * Маршрут установлення: включено в OpenClaw


## Інтерфейс

providers: qwen, qwencloud, modelstudio, dashscope; contracts: mediaUnderstandingProviders, videoGenerationProviders
