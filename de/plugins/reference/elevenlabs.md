# Elevenlabs-Plugin

Fügt Unterstützung für Provider zur Medienanalyse hinzu. Fügt Unterstützung für Provider zur Echtzeittranskription hinzu. Fügt Unterstützung für Text-to-Speech-Provider hinzu.

## Distribution

  * Paket: `@openclaw/elevenlabs-speech`
  * Installationsroute: in OpenClaw enthalten


## Oberfläche

Verträge: mediaUnderstandingProviders, realtimeTranscriptionProviders, speechProviders
