# Skill Workshop Plugin

Фіксує повторювані робочі процеси як Skills робочого простору, з очікуванням перевірки, безпечними записами та оновленням підказки Skill.

## Розповсюдження

  * Пакет: `@openclaw/skill-workshop`
  * Маршрут встановлення: входить до OpenClaw
