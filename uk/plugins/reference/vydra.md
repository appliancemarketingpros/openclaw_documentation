# Vydra Plugin

Додає до OpenClaw підтримку постачальника моделей Vydra.

## Поширення

  * Пакет: `@openclaw/vydra-provider`
  * Маршрут встановлення: входить до складу OpenClaw


## Інтерфейс

providers: vydra; contracts: imageGenerationProviders, speechProviders, videoGenerationProviders
