# Qwen Plugin

OpenClaw에 Qwen, Qwen Cloud, Model Studio, DashScope 모델 제공자 지원을 추가합니다.

## 배포

  * 패키지: `@openclaw/qwen-provider`
  * 설치 경로: OpenClaw에 포함됨


## 표면

제공자: qwen, qwencloud, modelstudio, dashscope; 계약: mediaUnderstandingProviders, videoGenerationProviders
