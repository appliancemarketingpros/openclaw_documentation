# Plugin MiniMax

Ajoute à OpenClaw la prise en charge des fournisseurs de modèles MiniMax et MiniMax Portal.

## Distribution

  * Package : `@openclaw/minimax-provider`
  * Chemin d’installation : inclus dans OpenClaw


## Surface

fournisseurs : minimax, minimax-portal ; contrats : imageGenerationProviders, mediaUnderstandingProviders, musicGenerationProviders, speechProviders, videoGenerationProviders, webSearchProviders
