# Plugin File Transfer

Ambil, cantumkan, dan tulis file pada node yang dipasangkan melalui perintah node khusus. Melewati pemotongan stdout bash dengan menggunakan base64 melalui node.invoke untuk biner hingga 16 MB.

## Distribusi

  * Paket: `@openclaw/file-transfer`
  * Rute instalasi: disertakan dalam OpenClaw
