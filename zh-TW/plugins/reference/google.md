# Google Plugin

為 OpenClaw 新增 Google、Google Gemini CLI、Google Vertex 模型提供者支援。

## 發佈

  * 套件：`@openclaw/google-plugin`
  * 安裝路徑：隨 OpenClaw 內含


## 介面

提供者：google、google-gemini-cli、google-vertex；契約：imageGenerationProviders、mediaUnderstandingProviders、memoryEmbeddingProviders、musicGenerationProviders、realtimeVoiceProviders、speechProviders、videoGenerationProviders、webSearchProviders
