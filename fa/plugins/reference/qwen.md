# Plugin ‏Qwen

پشتیبانی از ارائه‌دهنده‌های مدل Qwen، Qwen Cloud، Model Studio و DashScope را به OpenClaw اضافه می‌کند.

## توزیع

  * بسته: `@openclaw/qwen-provider`
  * مسیر نصب: در OpenClaw گنجانده شده است


## سطح

providers: qwen، qwencloud، modelstudio، dashscope؛ contracts: mediaUnderstandingProviders، videoGenerationProviders
