# Plugin StepFun

Добавляет поддержку провайдера моделей StepFun и StepFun Plan в OpenClaw.

## Распространение

  * Пакет: `@openclaw/stepfun-provider`
  * Способ установки: npm; ClawHub: `clawhub:@openclaw/stepfun-provider`


## Интерфейс

providers: stepfun, stepfun-plan
