# Plugin Elevenlabs

เพิ่มการรองรับผู้ให้บริการการเข้าใจสื่อ เพิ่มการรองรับผู้ให้บริการการถอดเสียงแบบเรียลไทม์ เพิ่มการรองรับผู้ให้บริการแปลงข้อความเป็นเสียง

## การเผยแพร่

  * แพ็กเกจ: `@openclaw/elevenlabs-speech`
  * เส้นทางการติดตั้ง: รวมอยู่ใน OpenClaw


## พื้นผิว

สัญญา: mediaUnderstandingProviders, realtimeTranscriptionProviders, speechProviders
