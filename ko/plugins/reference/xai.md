# xAI Plugin

OpenClaw에 xAI 모델 제공자 지원을 추가합니다.

## 배포

  * 패키지: `@openclaw/xai-plugin`
  * 설치 경로: OpenClaw에 포함됨


## 노출 영역

제공자: xai; 계약: imageGenerationProviders, mediaUnderstandingProviders, realtimeTranscriptionProviders, speechProviders, tools, videoGenerationProviders, webSearchProviders
