# Plugin de transferencia de archivos

Obtén, enumera y escribe archivos en nodos emparejados mediante comandos dedicados de nodo. Evita el truncamiento de `stdout` de bash usando base64 sobre node.invoke para binarios de hasta 16 MB.

## Distribución

  * Paquete: `@openclaw/file-transfer`
  * Ruta de instalación: incluido en OpenClaw
