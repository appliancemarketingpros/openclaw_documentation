# IRC Plugin

OpenClaw 메시지를 보내고 받기 위한 IRC 채널 표면을 추가합니다.

## 배포

  * 패키지: `@openclaw/irc`
  * 설치 경로: OpenClaw에 포함됨


## 표면

channels: irc
