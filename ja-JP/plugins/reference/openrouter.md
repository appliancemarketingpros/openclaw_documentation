# OpenRouter Plugin

OpenClaw に OpenRouter モデルプロバイダーのサポートを追加します。

## 配布

  * パッケージ: `@openclaw/openrouter-provider`
  * インストール経路: OpenClaw に含まれています


## サーフェス

providers: openrouter; contracts: imageGenerationProviders, mediaUnderstandingProviders, speechProviders, videoGenerationProviders
