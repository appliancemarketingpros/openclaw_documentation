# Plugin Anthropic Vertex

Plugin провайдера OpenClaw Anthropic Vertex для моделей Claude в Google Vertex AI.

## Распространение

  * Пакет: `@openclaw/anthropic-vertex-provider`
  * Способ установки: npm; ClawHub


## Поверхность

провайдеры: anthropic-vertex

## Claude Fable 5

Используйте `anthropic-vertex/claude-fable-5` там, где модель доступна в вашем регионе Google Cloud. Fable 5 всегда использует адаптивное мышление и по умолчанию задает усилие `high`. `/think off` и `/think minimal` используют усилие `low`, потому что модель не поддерживает отключение мышления.
