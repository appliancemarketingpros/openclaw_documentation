# Qwen Plugin

OpenClaw’a Qwen, Qwen Cloud, Model Studio ve DashScope model sağlayıcı desteği ekler.

## Dağıtım

  * Paket: `@openclaw/qwen-provider`
  * Kurulum yolu: OpenClaw’a dahildir


## Yüzey

providers: qwen, qwencloud, modelstudio, dashscope; contracts: mediaUnderstandingProviders, videoGenerationProviders
