# LLM Task Plugin

Універсальний LLM-інструмент лише з JSON для структурованих завдань, який можна викликати з робочих процесів.

## Розповсюдження

  * Пакет: `@openclaw/llm-task`
  * Маршрут встановлення: включено до OpenClaw
