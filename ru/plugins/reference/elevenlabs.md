# Plugin Elevenlabs

Добавляет поддержку провайдера понимания медиа. Добавляет поддержку провайдера транскрипции в реальном времени. Добавляет поддержку провайдера преобразования текста в речь.

## Распространение

  * Пакет: `@openclaw/elevenlabs-speech`
  * Способ установки: включен в OpenClaw


## Поверхность

контракты: mediaUnderstandingProviders, realtimeTranscriptionProviders, speechProviders
