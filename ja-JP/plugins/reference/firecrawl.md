# Firecrawl Plugin

エージェントから呼び出せるツールを追加します。Web フェッチプロバイダー対応を追加します。Web 検索プロバイダー対応を追加します。

## 配布

  * パッケージ: `@openclaw/firecrawl-plugin`
  * インストール経路: OpenClaw に含まれます


## サーフェス

contracts: tools, webFetchProviders, webSearchProviders
