# `openclaw clawbot`

Espace de noms d’alias hérité conservé pour la rétrocompatibilité.

Alias actuellement pris en charge :

  * `openclaw clawbot qr` (même comportement que [`openclaw qr`](</fr/cli/qr>))


## Migration

Préférez directement les commandes modernes de niveau supérieur :

  * `openclaw clawbot qr` -> `openclaw qr`
