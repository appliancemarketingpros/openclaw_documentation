# Firecrawl Plugin

에이전트가 호출할 수 있는 도구를 추가합니다. 웹 가져오기 공급자 지원을 추가합니다. 웹 검색 공급자 지원을 추가합니다.

## 배포

  * 패키지: `@openclaw/firecrawl-plugin`
  * 설치 경로: OpenClaw에 포함됨


## 노출 영역

contracts: tools, webFetchProviders, webSearchProviders
