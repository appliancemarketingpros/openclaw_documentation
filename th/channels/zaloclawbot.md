OpenClaw เชื่อมต่อกับ Zalo ClawBot ผ่าน Plugin ภายนอก `@zalo-platforms/openclaw-zaloclawbot` ที่แสดงอยู่ในแค็ตตาล็อก การเข้าสู่ระบบใช้รหัส QR ของ Zalo Mini App

## ความเข้ากันได้

เวอร์ชัน Plugin | เวอร์ชัน OpenClaw | npm dist-tag | สถานะ  
---|---|---|---  
0.1.x | >=2026.4.10 | `latest` | ใช้งานอยู่ / เบต้า  
  
## ข้อกำหนดเบื้องต้น

  * Node.js **> = 22**
  * ต้องติดตั้ง [OpenClaw](<https://docs.openclaw.ai/install>) แล้ว (`openclaw` CLI พร้อมใช้งาน)
  * บัญชี Zalo บนอุปกรณ์มือถือเพื่อสแกนรหัส QR สำหรับเข้าสู่ระบบ


## ติดตั้งด้วย onboard (แนะนำ)

เรียกใช้ตัวช่วยเริ่มต้นใช้งานของ OpenClaw แล้วเลือก **Zalo ClawBot** จากเมนูช่องทาง:

bashCopy code
[code]
    openclaw onboard
[/code]

ตัวช่วยจะติดตั้ง Plugin จากแค็ตตาล็อกทางการ (ตรวจสอบความถูกต้องแล้ว), แสดงรหัส QR สำหรับเข้าสู่ระบบในเทอร์มินัลโดยตรง และตั้งค่าช่องทางให้เสร็จเมื่อคุณสแกนด้วยแอป Zalo ไม่จำเป็นต้องใช้คำสั่งเพิ่มเติม

## การติดตั้งด้วยตนเอง

หากต้องการเพิ่มช่องทางไปยัง gateway ที่ผ่านการ onboard แล้ว ให้ทำตามขั้นตอนเหล่านี้:

### 1\. ติดตั้ง Plugin

bashCopy code
[code]
    openclaw plugins install "@zalo-platforms/openclaw-zaloclawbot@0.1.4"
[/code]

ใช้เวอร์ชันที่ปักหมุดไว้ตามที่แสดงด้านบนอย่างเคร่งครัด (ตรงกับรายการในแค็ตตาล็อกทางการ) เพื่อให้ OpenClaw ตรวจสอบแพ็กเกจกับแฮชความถูกต้องของแค็ตตาล็อกระหว่างการติดตั้ง

### 2\. เปิดใช้งาน Plugin ใน config

bashCopy code
[code]
    openclaw config set plugins.entries.openclaw-zaloclawbot.enabled true
[/code]

### 3\. สร้างรหัส QR และเข้าสู่ระบบ

bashCopy code
[code]
    openclaw channels login --channel openclaw-zaloclawbot
[/code]

สแกนรหัส QR ที่แสดงในเทอร์มินัลด้วยแอปมือถือ Zalo, ยอมรับข้อกำหนดการใช้งานภายใน Zalo Mini App และอนุญาตเซสชัน

### 4\. รีสตาร์ต Gateway

bashCopy code
[code]
    openclaw gateway restart
[/code]

## วิธีการทำงาน

ต่างจากช่องทาง Zalo สำหรับนักพัฒนามาตรฐานที่กำหนดให้คุณต้องลงทะเบียน Zalo Official Account (OA) ของคุณเองและวางข้อมูลประจำตัวนักพัฒนาแบบคงที่ Zalo ClawBot ทำงานเป็น **ผู้ช่วยส่วนตัวที่ผูกกับเจ้าของ** โดยใช้โครงสร้างพื้นฐานทางการร่วมกัน:

  1. **การเริ่มต้นใช้งานที่ปลอดภัย:** รหัส QR จะนำไปยัง Zalo Mini App ที่ปลอดภัย ซึ่งผูกบอทส่วนตัวที่จัดเตรียมใหม่ภายใต้ OA ทางการที่ใช้ร่วมกันเข้ากับ Zalo User ID ของคุณโดยตรง
  2. **ความเป็นส่วนตัวที่ผูกกับเจ้าของ:** ตามการออกแบบ บอทถูกจำกัดให้สื่อสารกับเจ้าของของมัน _เท่านั้น_ ข้อความจากผู้ใช้อื่นจะถูกทิ้งที่ระดับแพลตฟอร์ม ทำให้การเชื่อมต่อเป็นส่วนตัวและปลอดภัย
  3. **เส้นทาง API ทางการ:** Plugin ใช้ Zalo Bot Platform APIs แทน การทำงานอัตโนมัติผ่านเบราว์เซอร์หรือเว็บเซสชัน


## เบื้องหลังการทำงาน

Plugin Zalo ClawBot สื่อสารกับ Zalo APIs ผ่านลูปข้อความแบบ long-polling ที่คงอยู่ต่อเนื่อง เพื่อรักษารันไทม์ให้สะอาดและเบา:

  * การเชื่อมต่อ long-poll ใช้ endpoint `getUpdates`
  * Webhook ถูกปิดใช้งานตามค่าเริ่มต้นสำหรับการรัน Gateway บนเดสก์ท็อป/เทอร์มินัลในเครื่อง
  * ข้อความถูกประมวลผลฝั่งไคลเอนต์และแมปโดยตรงไปยังรันไทม์ agent ในเครื่องของคุณ


Plugin ภายนอกจัดการข้อมูลประจำตัวของบอทภายใต้ไดเรกทอรีสถานะของ OpenClaw ให้ถือว่าไดเรกทอรีนั้นเป็นข้อมูลอ่อนไหว และรวมไว้ในนโยบายควบคุมการเข้าถึงและ การสำรองข้อมูลเดียวกันกับสถานะ OpenClaw ส่วนที่เหลือของคุณ

## การแก้ไขปัญหา

  * **หมดเวลาการเข้าสู่ระบบด้วย QR:** โทเค็นเข้าสู่ระบบ (`zbsk`) จะหมดอายุหลังจาก 5 นาทีด้วยเหตุผลด้านความปลอดภัย หากรหัส QR หมดอายุก่อนที่คุณจะสแกน เพียงเรียกใช้คำสั่งเข้าสู่ระบบอีกครั้งเพื่อสร้างรหัสใหม่
  * **Gateway โหลดไม่สำเร็จ:** ตรวจสอบให้แน่ใจว่าเวอร์ชันโฮสต์ OpenClaw ของคุณเป็น `2026.4.10` หรือสูงกว่า เวอร์ชันเก่ากว่าไม่รองรับบัญชีแยกประเภทการติดตั้ง npm-plugin ภายนอก
