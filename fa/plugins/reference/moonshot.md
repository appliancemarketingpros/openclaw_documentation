# Plugin Moonshot

## توزیع

  * بسته: `@openclaw/moonshot-provider`
  * مسیر نصب: همراه OpenClaw ارائه شده است


## سطح

providers: moonshot; contracts: mediaUnderstandingProviders, webSearchProviders
