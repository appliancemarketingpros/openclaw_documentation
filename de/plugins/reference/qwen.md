# Qwen Plugin

Fügt OpenClaw Unterstützung für Qwen, Qwen Cloud, Model Studio und DashScope als Modell-Provider hinzu.

## Distribution

  * Paket: `@openclaw/qwen-provider`
  * Installationsweg: in OpenClaw enthalten


## Schnittstelle

Provider: qwen, qwencloud, modelstudio, dashscope; Verträge: mediaUnderstandingProviders, videoGenerationProviders
