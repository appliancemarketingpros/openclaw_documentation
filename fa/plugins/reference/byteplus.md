# Plugin BytePlus

## توزیع

  * بسته: `@openclaw/byteplus-provider`
  * مسیر نصب: همراه با OpenClaw ارائه شده است


## سطح

providers: byteplus, byteplus-plan; contracts: videoGenerationProviders
