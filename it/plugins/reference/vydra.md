# Plugin Vydra

Aggiunge il supporto per il provider di modelli Vydra a OpenClaw.

## Distribuzione

  * Pacchetto: `@openclaw/vydra-provider`
  * Percorso di installazione: incluso in OpenClaw


## Interfaccia

provider: vydra; contratti: imageGenerationProviders, speechProviders, videoGenerationProviders
