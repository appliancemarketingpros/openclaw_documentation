# Web Readability プラグイン

ローカル HTML Web フェッチ応答から読みやすい記事コンテンツを抽出します。

## 配布

  * パッケージ: `@openclaw/web-readability-plugin`
  * インストール経路: OpenClaw に含まれています


## 表面

契約: webContentExtractors
