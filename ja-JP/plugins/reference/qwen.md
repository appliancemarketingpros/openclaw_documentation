# Qwen Plugin

Qwen、Qwen Cloud、Model Studio、DashScope のモデルプロバイダーサポートを OpenClaw に追加します。

## 配布

  * パッケージ: `@openclaw/qwen-provider`
  * インストール経路: OpenClaw に含まれています


## サーフェス

プロバイダー: qwen, qwencloud, modelstudio, dashscope; コントラクト: mediaUnderstandingProviders, videoGenerationProviders
