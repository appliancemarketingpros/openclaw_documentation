# Firecrawl Plugin

ابزارهای قابل فراخوانی توسط عامل را اضافه می‌کند. پشتیبانی از ارائه‌دهنده واکشی وب را اضافه می‌کند. پشتیبانی از ارائه‌دهنده جست‌وجوی وب را اضافه می‌کند.

## توزیع

  * بسته: `@openclaw/firecrawl-plugin`
  * مسیر نصب: همراه با OpenClaw ارائه می‌شود


## سطح

قراردادها: tools, webFetchProviders, webSearchProviders
