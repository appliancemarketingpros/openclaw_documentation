# Elevenlabs Plugin

Додає підтримку провайдера розуміння медіа. Додає підтримку провайдера транскрибування в реальному часі. Додає підтримку провайдера перетворення тексту на мовлення.

## Дистрибуція

  * Пакет: `@openclaw/elevenlabs-speech`
  * Маршрут встановлення: включено в OpenClaw


## Поверхня

контракти: mediaUnderstandingProviders, realtimeTranscriptionProviders, speechProviders
