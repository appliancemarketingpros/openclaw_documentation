# [TOOLS.md](<http://TOOLS.md>) \- Локальні нотатки

Skills визначають, _як_ працюють інструменти. Цей файл призначений для _ваших_ особливостей — речей, унікальних для вашого налаштування.

## Що тут має бути

Речі на кшталт:

  * назв і розташувань камер
  * SSH-хостів і псевдонімів
  * бажаних голосів для TTS
  * назв динаміків/кімнат
  * псевдонімів пристроїв
  * усього, що залежить від середовища

## Чому окремо?

Skills є спільними. Ваше налаштування — ваше. Якщо тримати їх окремо, ви зможете оновлювати Skills, не втрачаючи свої нотатки, і ділитися Skills, не розкриваючи свою інфраструктуру.

* * *

Додавайте все, що допомагає вам виконувати свою роботу. Це ваша шпаргалка.
