# Deepgram Plugin

Додає підтримку провайдера розуміння медіа. Додає підтримку провайдера транскрипції в реальному часі.

## Розповсюдження

  * Пакет: `@openclaw/deepgram-provider`
  * Шлях встановлення: включено до OpenClaw
