# Plugin نقل الملفات

اجلب الملفات واعرضها واكتبها على Node المقترنة عبر أوامر Node مخصصة. يتجاوز اقتطاع مخرجات bash stdout باستخدام base64 عبر node.invoke للملفات الثنائية حتى 16 MB.

## التوزيع

  * الحزمة: `@openclaw/file-transfer`
  * مسار التثبيت: مضمّن في OpenClaw

العقود: الأدوات
