# Vydra Plugin

OpenClaw に Vydra モデルプロバイダーのサポートを追加します。

## 配布

  * パッケージ: `@openclaw/vydra-provider`
  * インストール経路: OpenClaw に含まれます


## サーフェス

プロバイダー: vydra; コントラクト: imageGenerationProviders, speechProviders, videoGenerationProviders
