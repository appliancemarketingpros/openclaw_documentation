# Plugin انتقال فایل

دریافت، فهرست‌کردن و نوشتن فایل‌ها روی گره‌های جفت‌شده از طریق فرمان‌های اختصاصی گره. با استفاده از base64 روی node.invoke برای باینری‌های تا 16 MB، محدودیت کوتاه‌سازی stdout در bash را دور می‌زند.

## توزیع

  * بسته: `@openclaw/file-transfer`
  * مسیر نصب: همراه با OpenClaw ارائه می‌شود
