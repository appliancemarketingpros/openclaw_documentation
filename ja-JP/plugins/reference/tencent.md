# Tencent Plugin

OpenClaw に Tencent TokenHub モデルプロバイダーのサポートを追加します。

## 配布

  * パッケージ: `@openclaw/tencent-provider`
  * インストール経路: OpenClaw に含まれています


## サーフェス

providers: tencent-tokenhub
