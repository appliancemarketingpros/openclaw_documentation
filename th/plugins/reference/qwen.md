# Plugin Qwen

เพิ่มการรองรับผู้ให้บริการโมเดล Qwen, Qwen Cloud, Model Studio และ DashScope ให้กับ OpenClaw

## การเผยแพร่

  * แพ็กเกจ: `@openclaw/qwen-provider`
  * วิธีติดตั้ง: รวมอยู่ใน OpenClaw


## พื้นผิว

ผู้ให้บริการ: qwen, qwencloud, modelstudio, dashscope; สัญญา: mediaUnderstandingProviders, videoGenerationProviders
