# `openclaw clawbot`

เนมสเปซ alias แบบเดิมที่คงไว้เพื่อความเข้ากันได้ย้อนหลัง

alias ที่ยังรองรับในปัจจุบัน:

  * `openclaw clawbot qr` (พฤติกรรมเหมือนกับ [`openclaw qr`](</th/cli/qr>))


## การย้าย

แนะนำให้ใช้คำสั่งระดับบนสุดแบบสมัยใหม่โดยตรง:

  * `openclaw clawbot qr` -> `openclaw qr`
