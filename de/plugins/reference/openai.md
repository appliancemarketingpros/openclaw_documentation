# OpenAI-Plugin

Fügt OpenClaw Unterstützung für die Modell-Provider OpenAI und OpenAI Codex hinzu.

## Distribution

  * Paket: `@openclaw/openai-provider`
  * Installationsweg: in OpenClaw enthalten


## Schnittstelle

Provider: openai, openai-codex; Verträge: imageGenerationProviders, mediaUnderstandingProviders, memoryEmbeddingProviders, realtimeTranscriptionProviders, realtimeVoiceProviders, speechProviders, videoGenerationProviders
