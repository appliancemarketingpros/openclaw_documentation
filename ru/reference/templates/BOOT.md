# BOOT.md

Добавьте короткие явные инструкции о том, что OpenClaw должен делать при запуске (включите `hooks.internal.enabled`). Если задача отправляет сообщение, используйте инструмент сообщений, а затем ответьте точным беззвучным токеном `NO_REPLY` / `no_reply`.
