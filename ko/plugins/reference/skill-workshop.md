# Skill Workshop Plugin

반복 가능한 워크플로를 워크스페이스 Skills로 캡처하며, 보류 중인 검토, 안전한 쓰기, Skill 프롬프트 새로고침을 지원합니다.

## 배포

  * 패키지: `@openclaw/skill-workshop`
  * 설치 경로: OpenClaw에 포함됨

계약: 도구
