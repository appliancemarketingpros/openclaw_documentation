# Openshell Plugin

Бекенд пісочниці на базі OpenShell із дзеркальованими локальними робочими просторами та виконанням команд через SSH.

## Поширення

  * Пакет: `@openclaw/openshell-sandbox`
  * Маршрут установлення: включено до OpenClaw
