ModelsProviders

NovitaAI — это поставщик размещенной AI-инфраструктуры с API моделей, совместимым с OpenAI. В OpenClaw это встроенный поставщик моделей, поэтому идентификатор поставщика — `novita`, учетные данные проходят через обычный поток авторизации моделей, а ссылки на модели выглядят как `novita/deepseek/deepseek-v3-0324`.

Используйте Novita, когда нужен размещенный доступ к маршрутам моделей с открытыми весами и сторонних моделей без запуска собственного сервера инференса. Встроенный каталог ориентирован на чат-модели, практичные для ходов агента, включая маршруты DeepSeek, Moonshot, MiniMax, GLM и Qwen, предоставляемые Novita.

Этот поставщик использует совместимый с OpenAI endpoint Novita. OpenClaw обрабатывает регистрацию поставщика, авторизацию, псевдонимы, нормализацию ссылок на модели и выбор базового URL; Novita управляет доступностью моделей в реальном времени, разрешениями аккаунта, ценами и лимитами частоты запросов.

## Настройка

Создайте ключ API на [novita.ai/settings/key-management](<https://novita.ai/settings/key-management>), затем выполните:

bashCopy code
[code]
    openclaw onboard --auth-choice novita-api-key
[/code]

Или задайте:

bashCopy code
[code]
    export NOVITA_API_KEY="<your-novita-api-key>" # pragma: allowlist secret
[/code]

## Значения по умолчанию

  * Поставщик: `novita`
  * Псевдонимы: `novita-ai`, `novitaai`
  * Базовый URL: `https://api.novita.ai/openai/v1`
  * Переменная окружения: `NOVITA_API_KEY`
  * Модель по умолчанию: `novita/deepseek/deepseek-v3-0324`


## Когда выбирать Novita

  * Вам нужен размещенный доступ к моделям с открытыми весами через API, совместимый с OpenAI.
  * Вам нужны маршруты DeepSeek, Kimi, MiniMax, GLM или семейства Qwen через один аккаунт поставщика.
  * Вам нужен еще один размещенный резервный путь наряду с OpenRouter, GMI, DeepInfra или прямыми API поставщиков.
  * Вы предпочитаете хостинг моделей на стороне поставщика вместо поддержки инфраструктуры vLLM, SGLang, LM Studio или Ollama.


Выбирайте прямого поставщика, когда вам нужны собственные параметры запросов поставщика или контракты поддержки. Выбирайте локального поставщика, когда модель должна работать на вашем собственном оборудовании или за вашей собственной сетевой границей.

## Модели

Встроенный каталог добавляет начальные идентификаторы распространенных маршрутов NovitaAI, включая:

  * `novita/moonshotai/kimi-k2.5`
  * `novita/minimax/minimax-m2.7`
  * `novita/zai-org/glm-5`
  * `novita/deepseek/deepseek-v3-0324`
  * `novita/deepseek/deepseek-r1-0528`
  * `novita/qwen/qwen3-235b-a22b-fp8`


Каталог служит отправной точкой для выбора моделей в OpenClaw. Ваш аккаунт, регион или текущий каталог Novita могут добавлять, удалять или ограничивать маршруты. Проверьте поставщика из CLI, прежде чем задавать долгосрочное значение по умолчанию:

bashCopy code
[code]
    openclaw models list --provider novita
[/code]

## Устранение неполадок

  * `401` или `403`: проверьте ключ на странице управления ключами Novita и повторно выполните `openclaw onboard --auth-choice novita-api-key`, если сохраненный профиль устарел.
  * Ошибки неизвестной модели: используйте точное значение `novita/<route-id>`, возвращенное командой `openclaw models list --provider novita`.
  * Медленные или неудачные маршруты: попробуйте другой маршрут модели Novita или задайте Novita как резервного поставщика для рабочих нагрузок, которые допускают различия между поставщиками.
