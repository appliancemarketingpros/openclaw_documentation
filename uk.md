## Що таке OpenClaw?

OpenClaw — це **самостійно розміщуваний gateway** , який з’єднує ваші улюблені чат-застосунки та канальні поверхні — вбудовані канали, а також пакетні або зовнішні Plugin каналів, як-от Discord, Google Chat, iMessage, Matrix, Microsoft Teams, Signal, Slack, Telegram, WhatsApp, Zalo та інші — з AI-агентами для програмування, такими як Pi. Ви запускаєте один процес Gateway на власному комп’ютері (або сервері), і він стає мостом між вашими застосунками для повідомлень та завжди доступним AI-асистентом.

**Для кого це?** Для розробників і досвідчених користувачів, які хочуть мати персонального AI-асистента, якому можна писати звідусіль, не відмовляючись від контролю над своїми даними й не покладаючись на розміщений сервіс.

**Чим це відрізняється?**

  * **Самостійне розміщення** : працює на вашому обладнанні за вашими правилами
  * **Багатоканальність** : один Gateway одночасно обслуговує вбудовані канали та пакетні або зовнішні Plugin каналів
  * **Орієнтовано на агентів** : створено для агентів програмування з використанням інструментів, сеансами, пам’яттю та маршрутизацією між кількома агентами
  * **Відкритий код** : ліцензія MIT, розвиток спільнотою


**Що вам потрібно?** Node 24 (рекомендовано) або Node 22 LTS (`22.16+`) для сумісності, API-ключ від вибраного провайдера та 5 хвилин. Для найкращої якості й безпеки використовуйте найпотужнішу доступну модель останнього покоління.

## Швидкий старт

* ### Установіть OpenClaw

bashCopy code
[code]
    npm install -g openclaw@latest
[/code]

* ### Виконайте початкове налаштування й установіть сервіс

bashCopy code
[code]
    openclaw onboard --install-daemon
[/code]

* ### Чат

Відкрийте інтерфейс керування в браузері та надішліть повідомлення:

bashCopy code
[code]
    openclaw dashboard
[/code]

Або підключіть канал ([Telegram](</uk/channels/telegram>) найшвидший) і спілкуйтеся з телефона.

Потрібне повне встановлення та налаштування для розробки? Див. [Початок роботи](</uk/start/getting-started>).

## Панель

Відкрийте браузерний інтерфейс керування після запуску Gateway.

  * Локальне значення за замовчуванням: <http://127.0.0.1:18789/>
  * Віддалений доступ: [вебповерхні](</uk/web>) і [Tailscale](</uk/gateway/tailscale>)

## Конфігурація (необов’язково)

Конфігурація розташована в `~/.openclaw/openclaw.json`.

  * Якщо ви **нічого не робите** , OpenClaw використовує пакетний бінарний файл Pi в режимі RPC із сеансами для кожного відправника.
  * Якщо хочете обмежити доступ, почніть із `channels.whatsapp.allowFrom` і правил згадування (для груп).


Приклад:

json5Copy code
[code]
    {  channels: {    whatsapp: {      allowFrom: ["+15555550123"],      groups: { "*": { requireMention: true } },    },  },  messages: { groupChat: { mentionPatterns: ["@openclaw"] } },}
[/code]
