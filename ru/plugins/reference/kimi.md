# Kimi Plugin

Добавляет поддержку поставщиков моделей Kimi и Kimi Coding в OpenClaw.

## Распространение

  * Пакет: `@openclaw/kimi-provider`
  * Маршрут установки: npm; ClawHub: `clawhub:@openclaw/kimi-provider`


## Интерфейс

поставщики: kimi, kimi-coding
