# Plugin do Qwen

Adiciona suporte aos provedores de modelo Qwen, Qwen Cloud, Model Studio e DashScope ao OpenClaw.

## Distribuição

  * Pacote: `@openclaw/qwen-provider`
  * Rota de instalação: incluído no OpenClaw


## Superfície

provedores: qwen, qwencloud, modelstudio, dashscope; contratos: mediaUnderstandingProviders, videoGenerationProviders
