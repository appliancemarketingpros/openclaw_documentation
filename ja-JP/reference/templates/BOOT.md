# [BOOT.md](<http://BOOT.md>)

起動時にOpenClawが何をすべきかについて、短く明確な指示を追加します（`hooks.internal.enabled` を有効にしてください）。 タスクがメッセージを送信する場合は、message toolを使用し、その後に正確な silent token `NO_REPLY` / `no_reply` で応答してください。
