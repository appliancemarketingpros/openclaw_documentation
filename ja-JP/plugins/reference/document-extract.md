# Document Extract Plugin

ローカルのドキュメント添付ファイルからテキストとフォールバック用のページ画像を抽出します。

## 配布

  * パッケージ: `@openclaw/document-extract-plugin`
  * インストール経路: OpenClaw に含まれています
