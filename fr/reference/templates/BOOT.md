# [BOOT.md](<http://BOOT.md>)

Ajoutez des instructions courtes et explicites sur ce qu’OpenClaw doit faire au démarrage (activez `hooks.internal.enabled`). Si la tâche envoie un message, utilisez l’outil message puis répondez avec le jeton silencieux exact `NO_REPLY` / `no_reply`.
