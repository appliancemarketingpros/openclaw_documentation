# Plugin Tokenjuice

Сжимает результаты инструментов exec и bash с помощью редьюсеров tokenjuice.

## Распространение

  * Пакет: `@openclaw/tokenjuice`
  * Способ установки: npm; ClawHub: `clawhub:@openclaw/tokenjuice`
