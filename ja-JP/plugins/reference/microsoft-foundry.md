# Microsoft Foundry Plugin

OpenClaw に Microsoft Foundry モデルプロバイダーのサポートを追加します。

## 配布

  * パッケージ: `@openclaw/microsoft-foundry`
  * インストール経路: OpenClaw に含まれています
