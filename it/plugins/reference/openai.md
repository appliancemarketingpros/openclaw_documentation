# Plugin OpenAI

Aggiunge a OpenClaw il supporto per i provider di modelli OpenAI e OpenAI Codex.

## Distribuzione

  * Pacchetto: `@openclaw/openai-provider`
  * Percorso di installazione: incluso in OpenClaw


## Superficie

provider: openai, openai-codex; contratti: imageGenerationProviders, mediaUnderstandingProviders, memoryEmbeddingProviders, realtimeTranscriptionProviders, realtimeVoiceProviders, speechProviders, videoGenerationProviders
