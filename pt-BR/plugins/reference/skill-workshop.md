# Plugin Skill Workshop

Captura fluxos de trabalho repetíveis como Skills do espaço de trabalho, com revisão pendente, gravações seguras e atualização do prompt da skill.

## Distribuição

  * Pacote: `@openclaw/skill-workshop`
  * Rota de instalação: incluído no OpenClaw
