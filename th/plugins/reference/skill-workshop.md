# Skill Workshop Plugin

บันทึกเวิร์กโฟลว์ที่ทำซ้ำได้เป็น Skills ของเวิร์กสเปซ พร้อมการตรวจทานที่รอดำเนินการ การเขียนอย่างปลอดภัย และการรีเฟรชพรอมป์ของ Skill

## การเผยแพร่

  * แพ็กเกจ: `@openclaw/skill-workshop`
  * เส้นทางการติดตั้ง: รวมอยู่ใน OpenClaw
