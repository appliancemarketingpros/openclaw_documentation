# `openclaw clawbot`

पिछली संगतता के लिए रखा गया लेगेसी उपनाम नेमस्पेस।

वर्तमान समर्थित उपनाम:

  * `openclaw clawbot qr` ([`openclaw qr`](</hi/cli/qr>) जैसा ही व्यवहार)


## माइग्रेशन

आधुनिक शीर्ष-स्तरीय कमांड सीधे उपयोग करें:

  * `openclaw clawbot qr` -> `openclaw qr`
