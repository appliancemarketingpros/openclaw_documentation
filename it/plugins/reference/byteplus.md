# BytePlus Plugin

Aggiunge il supporto dei provider di modelli BytePlus e BytePlus Plan a OpenClaw.

## Distribuzione

  * Pacchetto: `@openclaw/byteplus-provider`
  * Percorso di installazione: incluso in OpenClaw


## Interfaccia esposta

provider: byteplus, byteplus-plan; contratti: videoGenerationProviders
