# Plugin Qwen

Aggiunge a OpenClaw il supporto per i provider di modelli Qwen, Qwen Cloud, Model Studio e DashScope.

## Distribuzione

  * Pacchetto: `@openclaw/qwen-provider`
  * Percorso di installazione: incluso in OpenClaw


## Superficie

provider: qwen, qwencloud, modelstudio, dashscope; contratti: mediaUnderstandingProviders, videoGenerationProviders
