# OpenAI 插件

为 OpenClaw 添加 OpenAI、OpenAI Codex 模型提供商支持。

## 分发

  * 包：`@openclaw/openai-provider`
  * 安装路径：包含在 OpenClaw 中


## 暴露接口

提供商：openai、openai-codex；契约：imageGenerationProviders、mediaUnderstandingProviders、memoryEmbeddingProviders、realtimeTranscriptionProviders、realtimeVoiceProviders、speechProviders、videoGenerationProviders
